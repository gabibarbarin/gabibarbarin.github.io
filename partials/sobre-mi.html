  • Experiencia

    "Desarrollo web"

    En el rol de desarrollador web de momento como experiencia tengo:

    La mayor experiencia laboral fue el proyecto final para Alkemy. Realizamos una pagina web donde utilizamos React, Boostrap, Formik, Redux, Axios, SweetAlert, CKEditor, NodeJS, Sequelize, Express, JWT, Mercado Pago. La misma iba dirigida a para una ONG llamada Somos Mas. Esta pagina debia ser auto administrable post entrega sin la necesidad de ser programador, para eso diseñamos un backoffice al cual solo se podia ingresar siendo un usuario administrador en el que mediante formularios se podia editar la pagina.

    Challenge de ingreso a alkemy. Se debe crear una aplicación para llevar las cuentas donde se deba poder agregar nuevos ingresos, pero ademas editar y eliminar los existentes. Además debera ver un balance general de todos los ingresos y egresos en la cuenta. Debera contar con usuarios y la seguridad sera mediante JWT. Para el front utlice Javascript y para el back NodeJS con express.
